Local-first AI-video prompt ops

Video Prompt Lab

Turn one ShotSpec YAML into model-specific prompts for Sora, Veo, Runway, Kling, and Seedance — then generate a lint report and evaluation scorecard before you spend more credits.

Open GitHub repo View release Leave feedback

The workflow

  1. Write one structured ShotSpec YAML brief.
  2. Render prompts for multiple video models from the same source spec.
  3. Catch operational mistakes with a lint report.
  4. Compare outputs using a reusable scorecard instead of guessing.
idea -> ShotSpec YAML -> prompts -> lint -> scorecard -> iteration

Why it is useful

  • Reduces prompt rewrite time across models.
  • Preserves first-frame, camera, and motion intent.
  • Flags credit-wasting prompt omissions before generation.
  • Works locally without model API keys.

Included examples

  • Vertical product ad
  • Faceless YouTube Short
  • Mini-drama scene
  • China short-video template

Quick start

git clone https://github.com/YYH123123813/video-prompt-lab.git
cd video-prompt-lab
python3 -m pip install -r requirements.txt
python3 video_prompt_lab.py examples/product-ad.yaml \
  --models veo,sora,kling,seedance \
  --out outputs/product-ad
